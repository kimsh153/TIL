# KingFisher

### KingFisher는 원격저장소, 고유의 URL 주소를 가지고 있는 이미지를 앱 내에서 보여지게 해주는 라이브러리입니다.

### 특징
* 비동기식 이미지 다운로드 및 캐싱
* `URLSession`기반 네트워킹 또는 로컬 제공 데이터에서 이미지를 로드합니다.
* UIImageView에 편리하게 round 처리 기능

### 설치법

**Cocoapod**

```swift
pod 'KingFisher'
```
