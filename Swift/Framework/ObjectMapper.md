# ObjectMapper

### ObjectMapper?

ObjectMapper는 모델 객체를 JSON으로 또는 JSON에서 쉽게 변환할 수 있도록 하는 프레임워크입니다

### 이러한 기능이 있어요

* JSON to Objects 매핑
* Objects to JSON 매핑
* 중첩 객체(독립형, 배열 또는 딕셔너리)
* 매핑 중 사용자 지정 변환
* 구조체 지원
* 불변 지원

### 설치법

**Cocoapods**

```swift
pod 'ObjectMapper'
```
